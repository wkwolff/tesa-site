# TESA Website Phase 1 MVP - Raw Idea

## Project Overview

Build Phase 1 MVP website for **The Engineering and Science Academy (TESA)** - an aerospace education organization founded by Lt. Commander Diallo Wallace.

## Timeline

**Target Launch**: Ready for January 2026 legislative meetings

## Organization Context

- **Name**: The Engineering and Science Academy (TESA)
- **Focus**: Aerospace education
- **Founder**: Lt. Commander Diallo Wallace
- **Mission**: Provide aerospace education and training opportunities

## Technical Stack

### Core Technologies
- **Framework**: Next.js
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Hosting**: Railway

### Requirements

#### Accessibility
- **Standard**: WCAG 2.1 AA compliance
- All pages must meet accessibility guidelines
- Keyboard navigation support
- Screen reader compatibility
- Proper semantic HTML

#### SEO Foundation
- **Current Issue**: Zero visibility in search engines
- **Goal**: Fix SEO issues and establish foundation for search visibility
- Meta tags and descriptions
- Proper heading hierarchy
- Sitemap and robots.txt
- Open Graph and social media metadata

## Required Pages

### Core Pages
1. **Homepage**
   - Hero section introducing TESA
   - Mission and vision overview
   - Call-to-action elements
   - Featured programs/highlights

2. **About**
   - Organization history
   - Founder information (Lt. Commander Diallo Wallace)
   - Mission, vision, and values
   - Team information

3. **Program Overview**
   - Detailed program descriptions
   - Aerospace education offerings
   - Student opportunities
   - Success stories/testimonials

4. **Partnerships**
   - Current and potential partners
   - Collaboration opportunities
   - Partnership benefits
   - Contact for partnerships

5. **Contact**
   - Contact form
   - Organization contact information
   - Location/address (if applicable)
   - Social media links

6. **Blog Framework**
   - Blog listing page
   - Individual blog post template
   - Categories/tags support
   - Author information
   - Share functionality

## Current Status

- Existing site has zero search visibility
- Need to build foundation for legislative meetings
- Focus on professional presentation and accessibility
- Establish online presence for partnerships and outreach

## Success Criteria

- Professional, accessible website ready for January 2026
- WCAG 2.1 AA compliance across all pages
- SEO foundation in place with proper metadata
- Responsive design for all devices
- Fast page load times
- Easy content management capability
- Blog framework for ongoing updates
